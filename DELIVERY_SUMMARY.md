## ğŸ› TROUBLESHOOTING

### Issue: "FileNotFoundError: spam.csv"
**Fix:** Ensure spam.csv is in the same folder as scripts

### Issue: "No module named 'flask'"
**Fix:** Run `pip install -r requirements.txt`

### Issue: "Models not found"
**Fix:** Run `python train_and_save_model.py` first

### Issue: "Port 5000 in use"
**Fix:** Change port in app.py to 8080 or another available port

### Issue: NLTK data error
**Fix:** 
```python
import nltk
nltk.download('stopwords')
```
